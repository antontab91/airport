# Airport scoreboard

## QUICK START
In the project directory, you can run:

`npm i`

`npm start`

Runs the app in the development mode.<br />
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

The page will reload if you make edits.<br />

## The tech stack is:
+ HTML5
+ CSS3
+ Flexbox
+ Sass (Scss)
+ BEM methodology
+ JS ES6
+ ReactJS
+ React-router-dom
+ Redux
+ Redux-thunk
+ Moment.js
+ Reselect
